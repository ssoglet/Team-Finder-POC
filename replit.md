# 우리 학교 팀원 구하기 POC

## Overview
같은 학교 재학생 대상 팀원 구하기 플랫폼의 POC(Proof of Concept) 웹 애플리케이션입니다.
공모전, 대외활동 등을 위해 같은 학교 내에서 팀원을 쉽게 찾을 수 있는 서비스의 가능성을 시연합니다.

**핵심 플로우: 팀원 탐색 → 연결 → 소통**

## Tech Stack
- Python 3.11
- Streamlit (웹 UI 프레임워크)

## Project Structure
```
/
├── app.py                 # 메인 Streamlit 애플리케이션
├── .streamlit/
│   └── config.toml        # Streamlit 서버 설정
├── pyproject.toml         # Python 의존성
└── replit.md              # 프로젝트 문서
```

## Features

### 1. 탭 기반 네비게이션
- **팀원 찾기 커뮤니티**: 팀원 모집 게시판
- **팀원 검색**: 조건별 팀원 검색 및 메시지 전송
- **채팅**: 1:1 메시지 대화
- **본인 등록**: 프로필 등록 및 관리

### 2. 팀원 찾기 커뮤니티
- 팀원 모집 게시글 작성 (제목, 내용, 희망 학년/단과대/관심분야)
- 게시글 목록 카드 형태 표시
- 댓글 기능으로 참여 의사 표현

### 3. 팀원 검색
- 관심 분야, 단과대 기준 필터링
- 단과대별/관심분야별 인원 분포 차트
- 카드 형태 + 표 형태 결과 표시
- **메시지 보내기** 버튼으로 직접 연락

### 4. 채팅
- 좌측: 채팅방 리스트
- 우측: 선택한 채팅방 대화 화면
- 실시간 메시지 전송 (세션 기반)

### 5. 본인 등록
- 프로필 정보 입력 (이름, 학년, 단과대, 전공, 관심분야)
- 희망 활동 ON/OFF 토글
- 모든 활동 OFF 시 검색/커뮤니티에서 제외

### 6. 더미 데이터 생성
- 25명의 가상 재학생 데이터 자동 생성
- 3개의 샘플 게시글 자동 생성

## Running the App
```bash
streamlit run app.py --server.port 5000
```

## User Flow
1. "더미 데이터 생성" 버튼 클릭
2. "본인 등록" 탭에서 프로필 등록
3. "팀원 검색" 탭에서 조건 선택 및 팀원 탐색
4. "메시지 보내기"로 팀원에게 연락
5. "채팅" 탭에서 대화 확인
6. "팀원 찾기 커뮤니티"에서 게시글 작성/댓글
