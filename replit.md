# 우리 학교 팀원 구하기 POC

## Overview
같은 학교 재학생 대상 팀원 구하기 플랫폼의 POC(Proof of Concept) 웹 애플리케이션입니다.
공모전, 대외활동 등을 위해 같은 학교 내에서 팀원을 쉽게 찾을 수 있는 서비스의 가능성을 시연합니다.

**핵심 플로우: 팀원 탐색 → 연결 → 소통**

## Tech Stack
- Python 3.11
- Streamlit (웹 UI 프레임워크)

## Project Structure
```
/

├── pyproject.toml         # Python 의존성

```

## Features

### 1. 탭 기반 네비게이션
- **팀원 찾기 커뮤니티**: 공모전 정보 + 팀원 모집 게시판
- **팀원 검색**: 조건별 팀원 검색 및 메시지 전송
- **채팅**: 1:1 메시지 대화
- **본인 등록**: 프로필 등록 및 관리

### 2. 팀원 찾기 커뮤니티
- **공모전 정보 영역** (최상단 고정)
  - 교내/교외 공모전 구분 표시
  - 외부 링크 새 탭 열기
  - 공모전 사이트 바로가기 (접이식)
- 팀원 모집 게시글 작성 (제목, 내용)
- **희망 인원 수 설정** (1~10명)
- **인원별 개별 조건 설정** (학년/단과대/관심분야)
- 게시글 등록 후 자동으로 폼 닫힘
- 댓글 작성자 실명 표시 (단과대/전공 포함)

### 3. 팀원 검색
- **4가지 필터**: 학년, 전공, 관심 분야, 희망 활동
- **기본 필터**: 내 희망 활동과 겹치는 사람만 표시 (체크박스)
- **색상 차트**: Altair로 단과대별/관심분야별 분포 시각화
- 카드 형태 + 표 형태 결과 표시
- **메시지 보내기** 버튼에 학생 이름 표시

### 4. 채팅
- 좌측: 채팅방 리스트 (전공/학년/이름 형식)
- 최신 메시지만 미리보기 표시
- 우측: 선택한 채팅방 대화 화면
- 실시간 메시지 전송 (세션 기반)

### 5. 본인 등록
- 프로필 정보 입력 (이름, 학년, 단과대, 전공, 관심분야)
- 희망 활동 ON/OFF 토글
- 모든 활동 OFF 시 검색/커뮤니티에서 제외

### 6. 더미 데이터 생성
- 25명의 가상 재학생 데이터 자동 생성
- 3개의 샘플 게시글 자동 생성

## Running the App
```bash
streamlit run app.py --server.port 5000
```

## User Flow
1. "더미 데이터 생성" 버튼 클릭
2. "본인 등록" 탭에서 프로필 등록
3. "팀원 검색" 탭에서 조건 선택 및 팀원 탐색
4. "메시지 보내기"로 팀원에게 연락
5. "채팅" 탭에서 대화 확인
6. "팀원 찾기 커뮤니티"에서 게시글 작성/댓글
